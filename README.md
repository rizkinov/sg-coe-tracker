# sg-coe-tracker

COE Tracker — latest Singapore Certificate of Entitlement (COE) premiums for
all five categories, with change vs the previous bidding exercise, a two-year
premium trend chart, and per-category history.

## Data source

- [LTA COE bidding results via data.gov.sg](https://data.gov.sg/datasets/d_69b3380ad7e51aff3a7dcc84eba52b8a/view),
  fetched server-side and revalidated every 6 hours.

## Run locally

```
npm install && npm run dev
```

Part of [lifekit.sg](https://www.lifekit.sg).
